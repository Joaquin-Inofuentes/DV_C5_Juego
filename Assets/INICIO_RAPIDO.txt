┌─ FUNCIONAMIENTO ────────────────────────────────────────────────────────┐
│                                                                          │
│ 🔴 SOLDADO CAÍDO:                                                      │
│    • HP llega a 0 → Entra en DamagedState                             │
│    • GameObject "Vivo" se desactiva                                   │
│    • GameObject "Caido" se activa                                     │
│    • Indetectable por enemigos (ignoran completamente)                │
│    • No puede moverse ni disparar                                     │
│    • Líneas de debug a todos los aliados (azul)                       │
│                                                                          │
│ 👥 ALIADO SE ACERCA:                                                    │
│    • Si está dentro de 3 metros → línea se vuelve amarilla            │
│    • Si no está atacando/bajo fuego → puede revivir                   │
│    • Logs claros: "[CanReviveAlly] Soldado X PUEDE revivir a Y"      │
│                                                                          │
│ ⌨️  LÍDER REVIVA (Barra Espaciadora):                                   │
│    • Mantiene Spacebar por 3 segundos                                 │
│    • Barra amarilla se carga progresivamente                          │
│    • Si suelta o se aleja → barra baja rápido (rojo)                 │
│    • Completo → Soldado revive con 30% HP                            │
│    • Log: "[CompleteLeaderRevival] Soldado revivido por LIDER"       │
│                                                                          │
│ 🤖 ALIADOS IA REVIVEN (automático):                                     │
│    • Detectan caído cercano                                           │
│    • Si no están en combate → van a revivir automáticamente           │
│    • 3 segundos (sin input) → revivimiento completado                 │
│    • Log: "[CompleteRevival] Aliado ha revivido a caído"             │
│                                                                          │

│                                                                          │
│ Revivimiento en progreso:                                              │
│  [HandleRevivalInput] Líder comenzando a revivir a X                  │

│  [UpdateRevivalBar] Progreso: 50% | Color: Amarillo (subiendo)       │
│                                                                          │
│ Revivimiento completado:                                               │
│  [CompleteLeaderRevival] Líder Y ha revivido a X                      │
│  [CompleteLeaderRevival] Salud de X restaurada: 30/100                │
│  [CompleteLeaderRevival] Soldado X revivido por Líder Y               │
│  [Exit] Soldado X saliendo de estado CAIDO                            │
│                                                                          │
│ Revivimiento cancelado:                                                │
│  [HandleRevivalInput] Líder soltó barra espaciadora                   │
│  [UpdateRevivalBar] Progreso: 0% | Color: Rojo (bajando)             │

┌─ FLUJO COMPLETO DE EJEMPLO ─────────────────────────────────────────────┐
│                                                                          │
│ 1. Soldado "Guerrero" recibe 100 de daño (HP = 0)                     │
│    → [RecibirDano] Guerrero recibió 100. HP: 0/100                   │
│    → [EnterDamagedState] Guerrero entrando en estado CAIDO            │
│    → GameObject "Vivo" se desactiva                                   │
│    → GameObject "Caido" se activa                                     │
│                                                                          │
│ 2. Líder se acerca (distancia = 2.5m)                                  │
│    → Línea azul a Guerrero cambia a amarilla                          │
│    → [DrawLinesToAllies] Líder está cerca para revivir               │
│                                                                          │
│ 3. Líder mantiene Spacebar                                              │
│    → [HandleRevivalInput] Líder comenzando a revivir a Guerrero      │
│    → Barra amarilla aparece y comienza a subir                        │
│    → [HandleRevivalInput] Reviviendo a Guerrero. Progreso: 0.50/3s  │
│    → [UpdateRevivalBar] Progreso: 17% | Color: Amarillo              │
│                                                                          │
│ 4. A los 3 segundos:                                                    │
│    → [CompleteLeaderRevival] Líder ha revivido a Guerrero            │
│    → [CompleteLeaderRevival] Salud restaurada: 30/100                │
│    → Guerrero vuelve a SeguirFormacionState                           │
│    → GameObject "Caido" se desactiva                                  │
│    → GameObject "Vivo" se activa                                      │
│    → [Exit] Guerrero saliendo de estado CAIDO                         │
│                                                                          │
│ 5. Guerrero ahora está operativo nuevamente                            │
│    → Puede moverse, disparar, formar                                   │
│    → Logs: "[UnitController] Guerrero completamente CAIDO - ELIMINADO"│
│                                                                          │
